# Book Registry

Book Registry es una aplicación desarrollada con React, TypeScript y Vite que permite a los usuarios gestionar un registro de libros y explorar géneros literarios. Esta aplicación utiliza patrones de diseño para organizar y estructurar el código de manera eficiente.

## Características principales

- **Gestión de libros:** Permite agregar, editar y eliminar libros del registro.
- **Exploración de géneros literarios:** Incluye una funcionalidad para explorar diferentes géneros literarios como fantasía, ciencia ficción, misterio, entre otros.
- **Patrones de diseño:** Implementa patrones de diseño como el Factory Method para la creación de géneros literarios.

## Tecnologías utilizadas

- **React:** Biblioteca para construir interfaces de usuario.
- **TypeScript:** Superset de JavaScript que añade tipado estático.
- **Vite:** Herramienta de construcción rápida para proyectos web.

## Instalación y ejecución

Sigue estos pasos para instalar y ejecutar la aplicación en tu entorno local:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/JhoannPV/book-registry.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd book-registry
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
